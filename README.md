# Week01 Assignment

## 課題
関数 `fibonacci(n: int) -> int` を定義せよ。  
この関数は、n 番目のフィボナッチ数を返すものとする。

- 定義: F(0)=0, F(1)=1, F(n)=F(n-1)+F(n-2) for n>=2
- n は 0 以上の整数とする
